Formatting Elements in HTML

HTML provides various tags to format text and enhance the structure of your content. Here are some commonly used formatting elements:

  • <b> - Defines bold text.
  • <i> - Defines italicized text.
  • <u> - Defines underlined text.
  • <small> - Defines smaller text.
  • <mark> - Highlights text.
  • <del> - Represents deleted text (strikethrough).
  • <ins> - Represents inserted text (underlined).
  • <sub> - Defines subscript text.
  • <sup> - Defines superscript text.
  • <code> - Used to display code snippets or inline code.
  • <strong> - Represents important or strong emphasis.
  • <em> - Emphasizes text, typically italicized.
  • <blockquote>
    - Used for long quotations.
  • <pre>
    - Defines preformatted text, preserving spaces and line breaks.
